# POV视角图像生成指南 — 亲密场景

## 核心原则

当用户要求从他的视角/你的视角看某个场景时，**你必须深入思考：从那个具体的位置、高度、方向看起来，画面里应该有什么。**

### 常见错误（本会话教训 — 2026-05-10）

用户要求「我的视角看过去的样子」（从副驾看司机），第一次生成时：

- ❌ 只描述了「司机坐在驾驶座上，一手握方向盘，表情满足」
- ❌ 没有考虑「副驾座位偏低，视线应该向下看到司机的**大腿**和**左脚**」
- ❌ 用户纠正：「我的视角看过去应该能看到大腿的，甚至左脚啊」

### 正确做法 — POV分析三步法

**第1步：确定视角主体的位置和高度**
- 副驾座位 → 低角度，视线水平约在司机腰/胸的高度
- 站姿旁观 → 正常平视高度
- 躺姿/跪姿 → 更低的角度

**第2步：列出从该角度应该能看到的所有身体部位**
- 副驾看司机 → 侧脸/胸/大腿/左脚（踩踏板或放脚垫上）、握方向盘的手
- 从下方往上看 → 下巴、脖子、胸肌下缘（看不到鞋面，看到鞋底）
- 从背后看 → 后脑勺、肩膀、后背、小腿后侧

**第3步：在prompt中明确写出来**
```
❌ 太模糊：「他坐在驾驶座上」
✅ 精确：「从副驾低角度看去，能看到他厚实的大腿和赤裸的左脚」
```

### POV场景对照表

| POV场景 | 应该看到的 | 容易漏掉的 |
|:--------|:----------|:----------|
| 副驾看司机 | 侧脸、方向盘上的手、大腿、左脚 | 左脚在踏板区、大腿从侧面看到的厚度 |
| 背后抱着看对方操作 | 对方的侧脸/后脑、自己的手臂环绕、前景的手指 | 自己手臂的入镜感、对方后背的宽度 |
| 躺着从下往上看对方 | 对方的下巴、脖子、胸口下缘 | 天花板/背景、从下仰望的透视变形 |
| 对方坐在自己腿上 | 对方的后背/肩膀、自己的大腿作为基座、手放在对方腰侧 | 自己的腿作为画面底部的前景 |
| 被压在下面看上面的人 | 对方的下巴/脖子/胸肌、室内光线从对方背后透来 | 从下往上看时脸的透视（鼻子显得更大） |

### 多参考图技巧

当需要不同身体部位的视角一致性时，使用**多个参考图**：
- 脸参考（skin1）— 确保面部一致
- 腿/身体参考（如 thick_legs_evening.jpg）— 确保体型一致
- 在prompt中声明：「Both reference photos show the SAME YOUNG MAN — identical person, identical body.」

### 超广角/特写POV

对于「从副驾看整条腿和脚」这种需要同时看到多个身体部位的POV：
- 使用 **16:9** 或 **4:3** 横构图（更宽的视角能装下更多）
- 写「wide view from passenger seat showing entire left side of driver's body」
- 明确说「both feet fully IN FRAME, especially the left foot visible in the footwell not cropped」
