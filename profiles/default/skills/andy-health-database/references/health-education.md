## 🩸 糖尿病/血糖相关指标

### 1. 血糖类
| 指标 | 英文缩写 | 正常范围 | 糖尿病前期 | 糖尿病 | 说明 |
|------|----------|----------|-----------|--------|------|
| 空腹血糖 | FPG | < 6.1 mmol/L | 6.1-6.9 | ≥ 7.0 | 最基础的筛查 |
| 餐后2h血糖 | 2h-PG | < 7.8 mmol/L | 7.8-11.0 | ≥ 11.1 | 喝糖水后2小时测 |
| 随机血糖 | — | < 11.1 | — | ≥ 11.1 + 症状 | 任意时间采血 |

### 2. 糖化血红蛋白 HbA1c 👑 金标准
| 范围 | 判定 |
|------|------|
| < 5.7% | ✅ 正常 |
| 5.7-6.4% | ⚠️ 糖尿病前期 |
| ≥ 6.5% | ❌ 糖尿病 |

- 反映 **近2-3个月平均血糖**
- 每降1%，并发症风险降约30%

### 3. 胰岛功能
| 指标 | 正常参考 | 说明 |
|------|---------|------|
| 空腹胰岛素 (FINS) | 5-12 μIU/mL | 看基础分泌能力 |
| C肽 (C-Peptide) | 0.78-5.19 ng/mL | 更准，不受外源胰岛素干扰 |
| HOMA-IR | < 2.5 正常，> 2.5 提示抵抗 | = FPG × FINS ÷ 22.5 |

### 4. 并发症监控
| 指标 | 目的 |
|------|------|
| 血脂四项（TC/TG/HDL/LDL） | 代谢综合征评估 |
| 尿微量白蛋白/肌酐比 (UACR) | 早期肾损伤筛查 |
| 血肌酐/eGFR | 肾功能 |
| 眼底检查 | 糖尿病视网膜病变 |

### 5. 茶与糖尿病
- 茶多酚/儿茶素可提高胰岛素敏感性、抗氧化
- **辅助作用**，不能替代药物

## 🩺 血脂四项

| 指标 | 合适范围 | 说明 |
|------|---------|------|
| 总胆固醇 TC | < 5.2 mmol/L | 高了不好 |
| 甘油三酯 TG | < 1.7 mmol/L | 关联饮食和代谢 |
| HDL-C | > 1.0 (男) / > 1.3 (女) | **越高越好** — 好胆固醇 |
| LDL-C | < 3.4 mmol/L | **越低越好** — 坏胆固醇 |

## 🫁 肝功能

| 指标 | 正常参考 | 说明 |
|------|---------|------|
| ALT (谷丙转氨酶) | < 40 U/L | 肝细胞损伤标志 |
| AST (谷草转氨酶) | < 40 U/L | 肝/心/肌肉损伤 |
| GGT (γ-谷氨酰转肽酶) | < 60 U/L | 胆道/酒精性肝损伤 |

| 总胆红素 TBIL | < 21 μmol/L | 黄疸评估 |
| 总蛋白 TP | 65-85 g/L | 肝脏合成功能 |
