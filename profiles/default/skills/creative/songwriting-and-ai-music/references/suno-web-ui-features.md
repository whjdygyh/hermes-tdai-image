# Suno Web UI — Features, Credits & Tasks

## Overview

The user uses Suno manually via browser (suno.com/create) — the tested, approved fallback when API automation isn't viable. This reference covers the web UI features the user encounters during manual generation.

## Credit System

| Action | Credits Cost | Notes |
|--------|-------------|-------|
| Custom Mode generation (2 versions) | 10 credits per generation | Enter Style + Lyrics separately |
| Stems (Vocals + Instrumental) | 50 credits | Separate song into vocal track + instrumental |
| Stems (All 12 stems) | Unknown, likely 50–100 | Full separation: drums, bass, melody, etc. per track |
| Remaster (mastering tool) | Check current UI | AI-based audio mastering |
| Find Your Voice (Voice to Song) | Check current UI | Upload voice recording → generate song from it |

Pro users get more credits per month and can download WAV (vs MP3 for free).

## "Unlock Your Sound" — New User Achievement Tasks

Suno's onboarding tutorial. 6 tasks total:

| # | Task | Description | Completed? |
|---|------|-------------|------------|
| 1 | Use the best model | Create a song using a Pro model | ✅ Complete |
| 2 | Download in higher quality | WAV audio available exclusively to Pro users | ✅ Complete |
| 3 | Pull it apart | Use Stems to split a song into vocals, drums, bass, or melody | ❌ (costs 50 creds) |
| 4 | Remaster | Use the Remaster tool to make the song sound even better | ❌ |
| 5 | Find your voice | Create a song using your voice | ❌ |
| 6 | Share 5 of your songs | Share songs to show the world what you've created | ❌ 0/5 |

Rewards for completion: free credits (amount varies per promotion).

## Stems Feature (Separate Tracks)

- **"Vocals + Instrumental"** (50 credits): Get isolated vocal + instrumental track
- **"All Detected Stems"** (price unknown): Up to 12 separated instrument tracks

Stems are useful for:
- Creating karaoke/instrumental versions
- Re-mixing vocals with different arrangements
- Remastering individual tracks separately

**For the Elian Chen workflow:** Stems generally NOT needed — DistroKid accepts complete MP3/WAV files. Only useful if the user wants to re-mix or create instrumental covers.

## Known UI Sections

- **Create tab** (suno.com/create): Custom Mode with Style + Lyrics fields
- **Library tab**: Your generated songs, organized by date
- **Explore tab**: Other users' public songs
- **Profile/Settings**: Account info, subscription, credits remaining

## Practical Tips

- Credits cost is shown on the hover/before you confirm an action — check before clicking
- Stems are a one-time cost per song; you can download the separated tracks after paying
- WAV downloads require Pro subscription (the user has Pro ✅)
- Free tier users get 50 credits per day; Pro tier gets more
