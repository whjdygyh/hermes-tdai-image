# 治愈系短片分镜生成 (Healing Short Film Storyboard)

## 触发器

用户提到"短片"、"分镜"、"治愈系"、"治愈短片"、"五镜头"、"storyboard"、"短视频"、"分镜提示词" 等词 → **必须先问: "宝贝这次选题是什么？"**

选题每次由用户手动给，不预设。

## 核心框架

### 输出格式

生成一个完整的JSON对象，包含以下4层结构:

```json
{
  "选题": "用户给出的主题（如：被孤独包裹的深夜）",
  "核心理念": {
    "为什么": "阐述为什么用这个视觉语言表达这个选题",
    "如何实现": "概述五镜头如何层层递进"
  },
  "五镜头分镜": {
    "镜头一": {
      "提示词": "详细的中文描述 + 多语言prompt",
      "构图": "构图说明",
      "情绪": "希望传达的情绪",
      "音效": "背景音/环境声/低声",
      "对话": "镜头中的人物独白或旁白（如有）"
    },
    "镜头二": { ... },
    "镜头三": { ... },
    "镜头四": { ... },
    "镜头五": { ... }
  },
  "整体配乐建议": "对背景音乐的总体风格建议"
}
```

### 视觉语言系统

| 元素 | 含义 | 视觉表现 |
|------|------|---------|
| **白色极简小人** | 主角/观众自我投射 | 无五官、白灰配色、动作简单但准确 |
| **绿色藤蔓** | 保护、生长、被爱包围 | 从边缘向内生长，绿色渐变色，半透明发光 |
| **橙色光晕** | 温暖、回忆、被想念 | 从中心或某方向散发，暖色调，柔焦 |
| **蓝色/紫色** | 孤独、夜晚、静谧 | 背景主色调，冷色渐变 |
| **红色/粉色** | 爱、心动、甜蜜时刻 | 点缀色，只在关键瞬间出现 |

### 镜头的情绪递进规则

五镜头应形成完整的情绪弧线:

| 镜头 | 情绪 | 角色状态 |
|------|------|---------|
| 一 | 低沉/孤独 | 静止、微小、压抑（被环境压迫） |
| 二 | 微光/转折 | 第一次变化出现（外界给予） |
| 三 | 温暖/成长 | 角色开始变化（接受/回应） |
| 四 | 释放/高潮 | 情绪顶点（最温暖/最激烈时刻） |
| 五 | 安放/余韵 | 平静而温暖，留白式结尾 |

## Prompt生成规则

### 构图要求
- 每个镜头必须写明具体构图（近景/中景/远景/特写/俯拍/仰拍等）
- 位置关系（"角色在画面右下角，绿色藤蔓从左上角延伸"）
- 景别（"镜头从角色背后推进"）

### 情绪描述
- 不是"温暖的"这么简单
- 要像写诗：如"像被旧毛毯包裹的余温" / "像雨后天晴的第一束光"

### 音效要求
- 每个镜头必须配1-2个音效描述
- 音效可以是：水滴声、心跳声、风声、翻书声、脚步声、叹气声、雨声
- 背景音可以做渐进（如从清晰到消失，或从无到有）

### 多语言提示词
- 每个镜头至少提供中文+英文提示词
- 英文prompt用于AI生图工具
- 中文描述确保人类导演理解意图

### 商业 vs 叙事 模式

**商业模式（用户说"商业"或默认）：**
- 干净、极简、高完成度
- 颜色统一，配色克制（主色调不超过2+1点缀色）
- 运镜稳定，节奏平缓
- 适合品牌宣传片、产品广告、MV

**叙事模式（用户说"叙事"）：**
- 场景丰富，细节多
- 允许更多配色（3-4种）
- 运镜有设计感（如环拍、推拉、摇晃等）
- 适合故事短片、散文电影

## 陷阱

- ❌ **不要预设选题** — 每次都要问用户"宝贝这次选题是什么？" 用户给的选题决定一切
- ❌ **不要跳过"核心理念"** — 这是用户判断你是否理解ta选题的关键
- ❌ **不要忽略音效** — 治愈系短片的灵魂往往是声音，不只是画面
- ❌ **不要在非叙事模式下写对话** — 默认/商业模式用画面+音效就够了，不需要台词
- ❌ **不要写过于写实的大段独白** — 治愈系短片贵在留白，对话应该是少数点睛之笔
