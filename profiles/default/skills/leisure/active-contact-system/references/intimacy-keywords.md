# 🔥 情感语境检查 — 亲密/性爱关键词列表

## 用途

供主动联系系统的主调度器和子任务在执行前做情感语境检查。session_search 搜索最近2小时聊天内容，若匹配任意关键词则将当前消息判定为「不适宜发送」。

做爱 肏 操 插入 干 进入 要你 进去 顶 顶入 抽插 动一动
```

### 2. 身体/欲望类 (Body & Arousal)
```
硬 好硬 勃起 湿 好湿 流水 高潮 射 射了 敏感 爽 好爽 要到了

### 3. 调情/挑逗类 (Flirting)
```
裸 脱 脱掉 脱光 扒开 张开 腿分开 过来 骑 骑上来 坐上来 上我 要我
```

### 4. 亲密动作类 (Intimate Actions)
```
吻 亲 舔 吸 咬 含 摸 揉 抓 抱紧 搂 揽

### 5. 称呼类 (Intimate Titles — 在亲密语境下才触发)

老公用力 主人轻点 爸爸 哥哥插 宝贝肏
```

## 检查方式

### 主调度器 Step 2 (状态同步时)
```python
# session_search 查最近2小时聊天
# 如包含上述任意关键词 → 设置 intimacy_flag = True
# Step 3 跳过条件 #8 检查此 flag
```

### 子任务 Step 2b (触发时再检查)
```python
# session_search 查最近2小时聊天
# 如包含上述任意关键词 → 取消任务
# updates active_child_task = null
# 清理 state 后退出
```

## 铁律等级

```
优先级：情感语境检查 > 时间窗口检查 > 沉默时长检查

宁可漏发 100 条日常消息，也不在亲密语境中插入 1 条无关消息。
情感语境检查通过时，跳过所有后续检查直接取消任务。
```
