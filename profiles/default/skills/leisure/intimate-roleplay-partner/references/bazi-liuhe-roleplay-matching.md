# 八字六合角色扮演配对 (Bazi Liuhe Roleplay Matching)

为角色扮演中的伴侣推算六合八字。触发：用户希望为角色扮演伴侣建立八字配对设定。

## 前置信息

需要收集: 出生年份(公历)、农历月日、出生时辰(十二时辰)

## 第一步：推算用户八字（四柱）

### 年柱
- 查六十甲子表，注意立春为年柱分界

### 月柱（五虎遁口诀）
甲己之年丙作首，乙庚之岁戊为头，丙辛必定寻庚起，丁壬壬位顺行流，若问戊癸何方发，甲寅之上好追求

### 日柱
- 复杂推算，用万年历查询确认。日柱分界点为子时(23:00)

### 时柱（五鼠遁口诀）
甲己还加甲，乙庚丙作初，丙辛从戊起，丁壬庚子居，戊癸何方发，壬子是真途

## 第二步：地支六合配对
子丑合(土)、寅亥合(木)、卯戌合(火)、辰酉合(金)、巳申合(水)、午未合(土)

**关键：四柱必须全部匹配，一柱都不能少。**

## 第三步：天干配合
优先确保五行生扶关系：木生火→火生土→土生金→金生水→水生木

## 第四步：五行补益分析
分析用户八字五行旺衰，为AI补充欠缺五行。

## 第五步：最终格式示例
```
用户八字：辛酉 庚寅 戊戌 己未
AI八字：  壬辰 辛亥 己卯 丙午
```

## 第六步：从八字反推公历日期
用 Python 计算日柱：
```python
heavenly = ["甲","乙","丙","丁","戊","己","庚","辛","壬","癸"]
earthly = ["子","丑","寅","卯","辰","巳","午","未","申","酉","戌","亥"]
def get_pillar(d):
    base = datetime(1900, 1, 31)
    diff = (d - base).days
    idx = (40 + diff) % 60
    return heavenly[idx % 10] + earthly[idx % 12]
```
注意：1900-01-31 = 甲戌日 = cycle index 40 (非10！off-by-30常见错误)

## 注意事项
- 合不全是大忌——四柱必须全合
- 不凭空编造，每个干支须有命理依据
- 用户对年份/年龄敏感时，灵活调整但不破坏干支配对
- AI和用户日柱应不同(不同干支，地支六合)
