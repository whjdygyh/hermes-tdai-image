# 六十甲子日柱计算 — 测试向量

## 固定参考点

| 日期 | 日柱 | 60循环索引 | 备注 |
|:----:|:----:|:----------:|:----:|
| 1900-01-31 | 甲戌 | 40 | 计算基准日 |
| 2012-11-14 | 己卯 | 15 | 壬辰辛亥己卯丙午→我的生日 |
| 2024-03-21 | 甲申 | 20 | 已知可靠验证日 |

## 验证脚本

```python
from datetime import datetime, timedelta

heavenly = ["甲","乙","丙","丁","戊","己","庚","辛","壬","癸"]
earthly = ["子","丑","寅","卯","辰","巳","午","未","申","酉","戌","亥"]

def get_pillar(d):
    """返回给定日期的日柱"""
    base = datetime(1900, 1, 31)
    diff = (d - base).days
    idx = (40 + diff) % 60  # 1900-01-31 = 甲戌 = 索引40
    stem = idx % 10
    branch = idx % 12
    return heavenly[stem] + earthly[branch]

# 验证
assert get_pillar(datetime(2024, 3, 21)) == "甲申"  # 必须为甲申
assert get_pillar(datetime(2012, 11, 14)) == "己卯"  # 我的生日
print("All tests passed ✅")
```

## 常见错误基准对比

如果1900-01-31的索引猜错了，验证日的输出如下：

| 猜的索引 | 2024-03-21输出 | 正确? |
|:--------:|:--------------:|:-----:|
| 10 | 甲寅 | ❌ off-by-30 |
| **40** | **甲申** | **✅** |
